This is a universal object that combines (and will replace) presents-badge.html and series-badge.html. When calling the include use the series-badge-position parameter to specify the required positioning, "badge--story-item-placement", "badge--headline-placement badge--headline-placement--presents", "badge--aside-placement", or "badge--aside-placement badge--aside-placement--presents". Use the story-badge parameter to set what type of badge it should be, e.g. "presents", "series", or "bc-votes".
